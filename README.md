# مشروع التفريغ الصوتي وصياغة السطور المتزامنة بالذكاء الاصطناعي (WhisperX + FastAPI + Next.js)

مشروع متكامل ومعد للإنتاج الفوري (Production Ready) لتفريغ المقاطع الصوتية والفيديوهات بأي صيغة واستخراج النصوص التفسيرية والترجمات المتكاملة بثلاثة صيغ قياسية دقيقة ومطابقة للهامش الزمني (TXT, SRT, VTT) بدعم كامل للغة العربية ومجموعة واسعة من اللغات العالمية.

---

## ✨ المميزات البرمجية

- **التفريغ عالي الدقة (WhisperX Alignment):** استخدام نموذج فك التشفير الصوتي ومواءمته مع مرجع النطق اللغوي للحصول على دقة متكاملة للتوقيت على مستوى الكلمة الواحدة للوقوف الكلي والتلقائي.
- **استخراج فائق الأداء (FFmpeg Integration):** سحب تدفق موجات الصوت وتجزئته أوتوماتيكياً وتحويلها ومزامنتها إلى ربع طاقة العمل للحفاظ على سرعة الأداء.
- **تنوع الصيغ المحملة:**
  - **ملف TXT:** لتضمين القراءة أو لغرض التحرير والمراجعة والمستندات المطولة.
  - **ملف SRT:** لتجهيز تشغيل الترجمة أوفلاين في التطبيقات والمشغلات الشعبية كـ VLC.
  - **ملف VTT:** المعياري المناسب للويب والـ HTML5 Video players.
- **خيار الترجمة الفورية:** خيار متقدم لترجمة النصوص المستخرجة إلى أي لغة مستهدفة مع الحفاظ على الفواصل الزمنية الدقيقة وتماسك الجداول.
- **واجهة مستخدم احترافية:** واجهة سريعة الاستجابة، سهلة الاستخدام، مدمجة بالكامل بمؤشرات النسبة المئوية المباشرة لسير العمل، مع إمكانية التحرير اليدوي الفوري قبل التحميل وحفظ السجلات محلياً.
- **دعم المعالجة السريعة (CPU & CUDA/GPU):** التكيف الذكي مع العتاد المتوفر لتنفيذ التدفقات الحسابية بأعلى كفاءة ممكنة.

## 🚀 التشغيل بضغطة زر واحدة (Docker Quick Start)

لتشغيل كامل مكونات نظام الترجمة فوراً على الخادم أو محلياً دون الحاجة لتثبيت محلي لعناصر بايثون المعقدة:

```bash
docker-compose up --build
```

بعد انتهاء تشغيل الحاويات:
- **واجهة المستخدم:** `http://localhost:3000`
- **توثيق الـ API:** `http://localhost:8000/docs`

## 📖 تفاصيل ومعايير الأمان والأداء العالي

1. **إدارة استهلاك الذاكرة وحجم الملفات الكبيرة:** يتم تسييل الصوت واستخراجه كملف WAV مؤقت وحذفه مباشرة فور إنهاء الترجمة للحد من استهلاك قرص التخزين الصلب.
2. **مرونة الـ Multi-threading:** يتم تولي مهمة المزامنة والترجمة كخلفية برمجية مستقرة (FastAPI Background Tasks)، لتفادي انتهاء مهلة الاتصال (Timeout) وتمكين تتبع التقدم لحظياً عبر الـ Websocket أو الـ Polling.
3. **الدقة المتطابقة:** بفضل استخدام WhisperX (الذي يتغلب على ثغرات النموذج المرجعي الأصلي لـ Whisper في تجزئة الكلمات)، ستحصل على تطابق 100% بين ظهور العبارة ونطق المتحدث.

للمزيد من التعديلات التقنية وكيفية تفعيل الدعم لكرت الشاشة (GPU NVIDIA CUDA) يرجى مراجعة [دليل التشغيل التفصيلي المرفق هنا](/docs/GUIDE.md).
